# Spring Boot CRUD API

A simple CRUD REST API built using Spring Boot.

## 🚀 Tech Stack
- Java 25
- Spring Boot
- Spring Data JPA
- Maven
- MySQL

## ✨ Features
- Create Student
- Read Student
- Update Student
- Delete Student

## 📂 Project Structure

```
src/
 ├── controller
 ├── entity
 ├── repository
 └── service
```

## ▶️ Run the Project

Then open:

```
http://localhost:8080
```
